# 🌐 Mahan IP & Port Scanner

A powerful and simple Python tool to retrieve IP information and perform port scanning.
یک ابزار پایتونی قدرتمند و ساده برای دریافت اطلاعات آی‌پی و اسکن پورت‌ها.

## ✨ Features / قابلیت‌ها
*   **IP Info:** Get Country, City, ISP, and Organization details.
*   **Port Scanner:** Scan common ports to check their status (Open/Closed).
*   **User Friendly:** Clean terminal output with organized data.

*   **اطلاعات آی‌پی:** دریافت جزئیات کشور، شهر، سرویس‌دهنده اینترنت و سازمان.
*   **اسکن پورت:** بررسی پورت‌های رایج برای تشخیص باز یا بسته بودن آن‌ها.
*   **کاربری آسان:** خروجی تمیز و مرتب در ترمینال.

   git clone https://github.com/mduvdi89-dot/mahan_ip.git

   pip install requests

Run the script using Python:

اجرای اسکریپت با استفاده از پایتون:
